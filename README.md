# AuditPal Landing Page 


This is the standalone landing page for AuditPal - Decentralized AI Security.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Hero Section** - Compelling headline with holographic effects
- **Partner Logos** - Showcasing trusted partners
- **Benefits** - Key advantages of using AuditPal
- **Global Scale** - Intelligence dashboard with live metrics
- **Social Proof** - Testimonials and case studies
- **FAQ** - Frequently asked questions
- **Footer** - Navigation and contact info

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- TypeScript
